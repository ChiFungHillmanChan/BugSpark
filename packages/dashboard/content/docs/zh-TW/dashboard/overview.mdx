# 儀表板總覽

BugSpark 儀表板是您監控和管理錯誤報告的中心樞紐。

## 首頁

主儀表板一目了然地顯示四個關鍵指標：

- **錯誤總數** — 所有專案中錯誤報告的累計數量
- **未解決錯誤** — 狀態為 `new`、`triaging` 或 `in_progress` 的報告
- **今日已解決** — 過去 24 小時內標記為 `resolved` 的報告
- **平均解決時間** — 從報告建立到解決的平均時間

指標下方有兩個圖表提供視覺化洞察：

- **錯誤趨勢** — 顯示報告數量隨時間變化的折線圖
- **嚴重程度分佈** — `critical`、`high`、`medium` 和 `low` 報告的分佈

## 最近的錯誤

底部區段列出 5 個最新的錯誤報告，欄位包括追蹤 ID、標題、嚴重程度、狀態和建立日期。點擊任何列可查看完整報告。

## 導覽

側邊欄提供所有區段的快速存取：

| 區段 | 說明 |
|------|------|
| **儀表板** | 總覽指標和圖表 |
| **錯誤** | 包含表格和看板檢視的完整錯誤列表 |
| **專案** | 管理專案和 API 金鑰 |
| **設定** | 個人資料、外觀和語言 |
| **文件** | 本文件 |

## 專案切換

如果您有多個專案，請使用側邊欄中的下拉選單切換專案。報告和指標會篩選至所選專案。
